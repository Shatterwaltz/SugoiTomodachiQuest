# すごいともだちクエスト
Discord-based text adventure game for the ♥SuperFriends♥

Written with python3 using [Discord.py](https://github.com/Rapptz/discord.py)

## How to start the bot
win: `py.exe bot.py`

nix: `python3 bot.py`
